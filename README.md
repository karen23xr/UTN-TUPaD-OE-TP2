## 🚀 Descripción del Proyecto
Este repositorio contiene una solución automatizada en Python para el procesamiento, análisis y visualización de datos comerciales de una PyME. El desarrollo fue realizado de manera individual simulando un entorno profesional utilizando metodologías ágiles (Jira) y control de versiones distribuido (Git/GitHub) mediante el uso de ramas de trabajo dedicadas.

## 🛠️ Estructura del Repositorio
* `/datos`: Contiene el archivo fuente `dataset.csv` con los registros de ventas de la empresa.
* `/scripts`: Aloja el código fuente principal `analisis_datos.py` desarrollado en Python.
* `/resultados`: Carpeta que almacena las gráficas y reportes de rendimiento exportados de forma automática.

## 💻 Requisitos e Instalación
Para ejecutar el script en un entorno local o en Google Colab, se requiere Python 3.x y las siguientes librerías de procesamiento de datos:
```bash
pip install pandas matplotlib
```
